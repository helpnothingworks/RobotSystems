3针电池
=================

.. image:: img/3pin_battery.jpg

* **VCC**: 电池正极，在这里提供了两组 VCC 和 GND，用于增强电流并降低电阻。
* **中间引脚**: 平衡两节电池之间的电压，从而保护电池。
* **GND**: 电池负极。

这是由 SunFounder 定制的一款电池组，由两节 18650 电池组成，容量为 2000mAh。连接器为 XH2.54 3P，可直接插入 Robot HAT 进行充电。

**产品特点**

* 组成：锂离子电池
* 电池容量：2000mAh，14.8Wh
* 电池重量：90.8g
* 电芯数量：2
* 连接器：XH2.54 3P
* 过放电保护：6.0V

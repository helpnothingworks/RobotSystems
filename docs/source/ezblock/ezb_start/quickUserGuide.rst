快速用户指南
=====================================

EzBlock 3.2 的主要优化是将在线服务切换为离线模式，并新增项目文件的导入和导出功能。
届时，新用户注册通道将关闭，已注册用户登录后会收到提示，要求将项目转移至离线模式。
在线服务将在一段时间后关闭。

.. note::
    3.2 更新无需为树莓派重新烧录镜像。

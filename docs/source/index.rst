El PiCar-X es un coche robot autodirigido por IA para la plataforma Raspberry Pi, donde la Raspberry Pi actúa como el centro de control. 
El módulo de cámara de 2 ejes, el módulo ultrasónico y los módulos de seguimiento de líneas del PiCar-X ofrecen funciones como detección de colores/caras/señales de tráfico, 
evitación automática de obstáculos, seguimiento automático de líneas, etc.

PiCar-X tiene dos lenguajes de programación: Blockly y Python. No importa en qué idioma programes, encontrarás pasos detallados que te enseñarán desde la configuración de la Raspberry Pi hasta ejecutar el código de ejemplo relevante.

* :ref:`play_python`

    * Este capítulo es para quienes disfrutan programando en Python o desean aprender el lenguaje Python.
    * Para que PiCar-X funcione correctamente, primero debes instalar algunas bibliotecas.
    * En este capítulo se proporciona la configuración de la Raspberry Pi y el código de ejemplo para PiCar-X.
    * También se ofrece una APP - SunFounder Controller para permitirte controlar remotamente el PiCar-X desde tu dispositivo móvil.


* :ref:`play_ezblock`

    * En esta sección, utilizarás una APP basada en Blockly, Ezblock Studio, que, al igual que Scratch, te permite arrastrar y soltar bloques para mover el PiCar-X.
    * Es necesario reinstalar la tarjeta SD con el sistema operativo que proporcionamos, el cual incluye el entorno Ezblock preinstalado antes de programar. Se recomienda usar una tarjeta TF nueva o sin uso para esta sección.
    * Ezblock Studio está disponible para casi todo tipo de dispositivos, incluidos Mac, PC y Android.
    * Ezblock Studio es una buena opción si tienes entre 6 y 12 años, no tienes habilidades de programación o deseas probar rápidamente el PiCar-X.

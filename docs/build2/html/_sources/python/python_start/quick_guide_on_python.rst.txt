Pythonに関するクイックガイド
==============================

このセクションでは、Raspberry Pi OSのインストール、Raspberry Piへのwifiの設定、Raspberry Piへのリモートアクセスをして対応するコードを実行する方法を教えます。

Raspberry Piに慣れており、コマンドラインを開くことができる場合は、最初の3部分をスキップして、最後の部分だけを完了してください。

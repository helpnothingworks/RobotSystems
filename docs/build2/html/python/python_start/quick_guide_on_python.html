Pythonに関するクイックガイド¶

このセクションでは、Raspberry Pi OSのインストール、Raspberry Piへのwifiの設定、Raspberry Piへのリモートアクセスをして対応するコードを実行する方法を教えます。

Raspberry Piに慣れており、コマンドラインを開くことができる場合は、最初の3部分をスキップして、最後の部分だけを完了してください。

  • 必要なものは何ですか？
  • OSのインストール
  • Raspberry Piのセットアップ
  • すべてのモジュールをインストールする（重要）
  • I2Cインターフェースを有効にする（重要）
  • サーボ調整（重要）

動画

組み立て動画の6:25から8:48には、この章の詳しいチュートリアルもあります。動画の指示に直接従うことができます。

Note

  • 6:25から8:48のセクションのみをご覧ください。

  • 動画の残りの部分はPiCar-X 2.0についてのもので、お受け取りになった紙の組み立て指示と異なる組み立てステップが含まれている場合があります。正確さのために紙のマニュアルを参照してください。
